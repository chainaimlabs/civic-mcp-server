# civic-mcp-server
civic-mcp-server

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn package manager
- Git

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/chainaimlabs/civic-mcp-server.git
   cd civic-mcp-server
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Build the project**
   ```bash
   npm run build
   ```

4. **Run the server**
   ```bash
   npm start
   ```
